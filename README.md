# Basic Calculator
This is a basic calculator web application built using HTML, CSS, and JavaScript. It provides simple arithmetic operations such as addition, subtraction, multiplication, and division.

## **==> Features**
Addition
Subtraction
Multiplication
Division

## **Technologies Used**
HTML
CSS
JavaScript
